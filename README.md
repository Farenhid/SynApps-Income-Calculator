# SynApps Income Calculator

ابزار تک‌صفحه‌ای (بدون build) برای محاسبه **حقوق پایه ساعتی** بر اساس مدل تاب‌آوری و **پرداخت ماهانه** بر اساس عدد پایه، ضریب فردی (ماتریس ارزیابی) و ساعات کار.

## ویژگی‌ها

- **تب حقوق پایه ساعتی:** ورودی کف بقا، نرخ ایده‌آل بازار، تاب‌آوری فعلی و هدف؛ خروجی نرخ پایه با فرمول مدل تاب‌آوری.
- **تب ضریب و پرداخت ماهانه:** انتخاب نقش، امتیازدهی ۱۰ معیار در چهار بُعد، ضریب فردی، عدد پایه ساعتی، ساعات کار ماهانه و مبلغ نهایی.
- رابط **فارسی و RTL** با فونت Vazirmatn و Tailwind از CDN.
- لینک **منبع فرمول** در هر تب به داکیومنت‌های مرجع (Google Docs).

## اجرای محلی

فایل [`index.html`](index.html) را مستقیماً در مرورگر باز کنید، یا با یک سرور استاتیک ساده:

```bash
cd SynApps-Income-Calculator
python3 -m http.server 8080
```

سپس به آدرس `http://localhost:8080` بروید.

## فناوری

- HTML + React 18 (UMD) + Babel Standalone + Tailwind CDN

## فرمول‌ها (خلاصه)

**حقوق پایه ساعتی:**

`Base_Rate = W_min + (W_target - W_min) * min(1, R_actual / R_target)`

**پرداخت ماهانه:**

`Pay = Base_Rate × Multiplier × Hours`

ضریب فردی وقتی حداقل یک معیار امتیاز گرفته باشد: `Multiplier = 0.70 + (TotalScore × 0.01)`؛ در غیر این صورت `1.00`.

جزئیات و منطق کامل در داکیومنت‌های لینک‌شده داخل اپ قرار دارد.
